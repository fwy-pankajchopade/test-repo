# Saved Copilot Outputs

Save your Copilot Chat responses here during each bug fix or feature session.

## Naming Convention

```
outputs/BUG-123-fix.md       ← the fix proposed by Copilot Chat
outputs/BUG-123-tests.md     ← the tests proposed by Copilot Chat
outputs/BUG-123-review-context.md  ← auto-generated by run-review.sh
```

## Why Save Outputs?

- Traceability — know exactly what Copilot suggested vs what you applied
- Review — paste into review_ticket_prompt.md for pre-PR review
- Audit — useful for post-mortems and lessons-learned entries
